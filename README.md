# Relación Pico-Masa: Un Enfoque en la Inversión Estructural por Sexo en Pingüinos de Palmer

Este repositorio contiene un caso de estudio enfocado en la relación proporcional entre la masa corporal y las dimensiones del pico (longitud y profundidad) en tres especies de pingüinos de Palmer. El objetivo es evaluar si este índice de razón funciona como una herramienta matemática y no invasiva para la diferenciación sexual en campo.

## Resumen del Proyecto

El objetivo principal de esta investigación es evaluar la relación proporcional entre la masa corporal y las dimensiones del pico (longitud y profundidad) como una alternativa metodológica y no invasiva para la identificación del sexo en campo, reduciendo la necesidad de procedimientos invasivos y priorizando el bienestar animal.

- **Datos utilizados:** Dataset clásico *Palmer Penguins* (Gorman, Williams & Fraser, 2014).
- **Contexto:** Desarrollado durante el primer semestre de la Maestría en Ciencia de Datos de la Universidad Autónoma de Bucaramanga (UNAB).
